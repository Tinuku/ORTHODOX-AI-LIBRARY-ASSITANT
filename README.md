# ORTHODOX AI LIBRARY ASSISTANT
## Project Sovereign

### Overview
Development of a localized Large Language Model (LLM) specialized in Patristic theology. The project prioritizes data sovereignty and theological accuracy through local execution and QLoRA fine-tuning.

### Technical Specifications
- Platform: Lenovo LOQ 15
- GPU: NVIDIA GeForce RTX 4060 (8GB VRAM)
- RAM: 24GB/32GB DDR5 5600MT/s (Targeting 32GB)
- Storage: NVMe SSD

### Project Phases
1. Hardware Provisioning and Data Collection
2. Local Environment Setup (Ollama/PyTorch)
3. Model Fine-Tuning and Evaluation
4. Deployment of Interface and Guardrails

### Ethics and Compliance
This tool is intended for academic and research purposes. All outputs must be cross-referenced with primary sources.
